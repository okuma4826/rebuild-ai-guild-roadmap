# 学習管理スプレッドシート

Rebuild AI Guild の学習管理用テンプレートです。

## ダウンロード

- **[ギルド学習管理シート.xlsx](ギルド学習管理シート.xlsx)** — 正本ファイル

## Google スプレッドシートで使う

1. 上の xlsx をダウンロードする（GitHub ではファイル名をクリック → **Download**）
2. [Google ドライブ](https://drive.google.com) にアップロードする
3. ファイルを開き、**Googleスプレッドシートで開く** を選ぶ
4. タイトルを自分用の名前に変える

含まれるシート：`01_習慣設計` / `02_週間時間割` / `03_日々の記録` / `04_行動変容ワーク` / `05_週報月報`

詳しい手順は [第1章 02 学習管理スプシをコピーする](../../教材/第01章-明確な目標と習慣/02-学習管理スプシをコピーする.md) を参照してください。
